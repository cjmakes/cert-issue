Getting & Using A Cert


	# Obtain a signed certificate
	curl https://host/api/v1/{user}/certs/1 > .ssh/scale-cert.pub

	# Add config for the scale servers to your ssh config
	cat >> .ssh/config << EOF
	Host *
	  User {user}
	  CertificateFile ~/.ssh/scale-cert.pub
	EOF

How does this work

This service issues signed certificates to allow users to access Scale infrastructure.
